# docx4j-wisewe
需要JDK8及以上


## [文档导出](./docx4j-wisewe-output/README.md)

maven引入

```xml
<dependency>
    <groupId>cn.wisewe</groupId>
    <artifactId>docx4j-wisewe-output</artifactId>
    <version>见上最新版本</version>
</dependency>
```

gradle引入

```groovy
dependencies {
    implementation "cn.wisewe:docx4j-wisewe-output:见上最新版本"
}
```
